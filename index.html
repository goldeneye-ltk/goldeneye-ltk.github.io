GE Netplay LTK Cup Edition v1.3

GE Netplay LTK Cup Edition v1.3

NOTE: The Nintendo 64 Expansion Pak is required for this mod, otherwise it will display a black screen at boot.

Multiplayer enhancements:

  • Added new weapon sets, including randomized weapon sets (Casino Royale, Russian Roulette)
  • Every character is unlocked by default, players can use the same ones.
  • There are 12 new multiplayer levels. Every level is unlocked by default. Press 'next' on the level select screen for a second page multiplayer levels.
  • Bond Invisible cheat works in multiplayer
  • Four players allowed on every level
  • Independently selectable look up and down for each player
  • NITRA (No-spawnkills Intelligent Tournament Respawn Algorithm™) enables random spawns and prevents spawnkills
  • New spawns locations (Bunker 2, Temple, Archives and Stack)
  • Players start armed in multiplayer, unless disabled via a cheat
  • Tiny Bond cheat works in multiplayer
  • Weapon set customization screen. Press 'R' on current weapon set to customize.
  • Press 'R' on game length to customize lives, points and time limit.
  • Dual wield cheat for True Dual Wielding™, which allows you to dual wield whenever you find another of same weapon
  • No i-frames cheat (no invincibility when taking damage)
  • Dr. No cheat that removes bullet armor from maps
  • Extra handicaps (including an LTK handicap that behaves just like LTK)
  • The Man With the Golden Gun mode changed so points can only be scored using the Golden Gun
  • Soft lock screens can be exited by pressing 'L' (player 1's controller only)
  • "Disable fog" cheat that makes players fullbright on maps like Caves
  • A common configuration is loaded so save files don't have to be shared before starting the game

Multiplayer fixes:

  • Fixed camera disorientation when respawning
  • Fixed random player orientation when respawning
  • Fixed mines not detonating when another player is shooting
  • Fixed players surviving with zero health when shot in LTK mode (also known as "ghost health")
  • Fixed projectiles passing through the map on Cradle
  • Fixed the body hitboxes of five characters
  • Fixed the POV height for three female characters

General improvements:

  • Dynamic firing rate (the game will adjust the firing rate so guns fire at their normal rate when the game hits 60 FPS)
  • Dynamic LODs (low-detail LODs only appear when the game runs at 20 FPS or worse)
  • Stereo sound in single player (yeah, the OG game is mono)

Performance improvements:

  • Better framerate that can make it difficult to go back to the OG game when playing on real hardware.
  • If the music was completely disabled, it won't be processed when playing multiplayer, thus improving the framerate a bit.
  • Optimized Stand Tiles line of sight algorithm to be 5x faster. Reduces lag when many guards on screen.
  • Recompressed assets using a 4x faster algorithm (lz4). Reduces lag and frameskips, such as lag that occurs when switching weapons.
  • Reduced lag due to watch (watch now always runs at a minimum of 30 FPS, there is no framerate jittering when opening or closing the watch)
  • Various optimizations that make the game run at a better framerate such as code rewritten to make it more cache-friendly.

Single-player bug fixes:

  • Fixed Baron Samedi boss freezing
  • Fixed crate guards freezing on every level
  • No-gravity projectiles fix (Bunker 2)

Incomplete:

  • A dual-wielding player only drops a single copy of their gun upon dying
  • Anti-aliasing cheat isn't yet implemented
  • Four new multiplayer levels coming in the next version
  • More "meme" multiplayer cheats
  • Finer controls on custom match length screen
  • It's very likely that the spawn in the top room on Temple will be reversed to face the other direction
